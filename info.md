# EMS Balcony Solar Integration

## Overview
The EMS Balcony Solar integration allows users to monitor and control their solar energy systems directly from Home Assistant. This integration provides various sensor and binary sensor entities to track energy production, consumption, and system status.

## Features
- Real-time monitoring of solar energy production.
- Binary sensors for system status (e.g., operational, fault).
- User-friendly configuration through Home Assistant's UI.
- Localization support for multiple languages.

## Installation
1. Ensure you have Home Assistant installed and running.
2. Install the HACS (Home Assistant Community Store) if you haven't already.
3. Add the `ems_balcony_solar` integration through HACS by searching for it in the HACS store.
4. Follow the configuration instructions provided in the integration's settings.

## Usage
Once installed, you can add the `ems_balcony_solar` integration through the Home Assistant integrations page. After configuration, the integration will create various entities that you can use in your automations and dashboards.

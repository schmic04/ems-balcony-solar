# EMS Balcony Solar Integration

## Overview
The EMS Balcony Solar integration allows you to monitor and control your balcony solar system through Home Assistant. This integration provides various sensors and binary sensors to track the performance and status of your solar setup.

## Features
- Monitor solar energy production
- Track battery status
- Receive alerts for system issues

## Installation
1. Ensure you have Home Assistant installed and running.
2. Install the HACS (Home Assistant Community Store) if you haven't already.
3. Add the `ems_balcony_solar` integration through HACS:
   - Go to HACS in your Home Assistant sidebar.
   - Click on "Integrations".
   - Search for "EMS Balcony Solar" and install it.

## Configuration
After installation, you need to configure the integration:

1. Go to `Configuration` in Home Assistant.
2. Click on `Integrations`.
3. Find `EMS Balcony Solar` and follow the prompts to set it up.

## Usage
Once configured, the integration will create various entities in Home Assistant that you can use in your automations and dashboards. You can monitor the solar production, battery status, and receive notifications for any issues.
